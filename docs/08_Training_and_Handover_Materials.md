# Training and Handover Materials Outline

## 1. Training Modules
- **Module 1: Introduction to TB e-Tracker**: Overview, DHIS2 navigation, and data privacy principles.
- **Module 2: Facility Data Clerks**: Patient registration, screening, and basic follow-up entry.
- **Module 3: Clinicians and Lab Officers**: Diagnosis workflows, treatment initiation, and lab results logging.
- **Module 4: M&E Officers**: Dashboard interpretation, data quality checks, and cohort reporting.
- **Module 5: Mobile App Usage**: Offline data entry via DHIS2 Android Capture, synchronization, and troubleshooting.
- **Module 6: System Administration**: User management, metadata updates, and basic troubleshooting.

## 2. Job Aids and Quick Reference Guides
- 1-page "Cheat Sheet" for Patient Registration.
- Flowchart for TB/HIV Co-infection data entry.
- Troubleshooting guide for common sync errors.
